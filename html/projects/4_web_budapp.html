Création d'une web app :
BUD app

Voici la web app BUD app, réalisée en 2 jours .

J'ai ici décider de réaliser une web app de budget.
Étant étudiante j'utilise ce genre d'application au quotidien pour gérer mon budget et mes dépenses.


Pour ce projet j'ai utilisé :
- Ruby on Rails
- Bootstrap
 - CSS et HTML


Vous pouvez ici :
- vous inscrire, connectez, déconnectez
- créer de nouvelles dépenses ou entrées d'argent
- visualiser celle ci grâce à un graphique
- voir, modifier et supprimer chaques opérations
- trier vos dépenses par catégories


BUD app est hébergé sur Scalingo.
