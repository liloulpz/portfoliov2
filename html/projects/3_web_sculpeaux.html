Création d'un blog
Back-End

Voici le blog Sculp'eaux, réalisée en trois semaines .

Nous avons choisi le thème "Bordeaux à travers... les sculptures" .


Pour ce projet de back-office nous avons utilisé :
- PHP
- mySQL.


Nous avons réalisé les CRUD (create, update, delete) de chacunes des tables fournies (voir ici).
Pour qu'un administrateur puisse créer, modifier et supprimer facilement sans toucher une seule ligne de code,
un article, un commentaire, un membre et bien d'autres.


Le blog est hébergé sur le serveur de l'IUT Bordeaux Montaigne.
